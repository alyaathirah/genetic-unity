Detailed Description

An Academy is where Agent objects go to train their behaviors.

Currently, this class is expected to be extended to implement the desired academy behavior.

When an academy is run, it can either be in inference or training mode. The mode is determined by the presence or absence of a Communicator. In the presence of a communicator, the academy is run in training mode where the states and observations of each agent are sent through the communicator. In the absence of a communicator, the academy is run in inference mode where the agent behavior is determined by the Policy attached to it.
